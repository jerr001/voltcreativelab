## How It Works

When a potential customer clicks a sales rep's link with the `?ref=` parameter:

1. The referral code auto-populates in the form
2. When the form is submitted, the code is included in the email notification
3. The company tracking email shows which sales rep referred the lead

## Adding New Sales Reps

To add a new sales rep:

1. Choose their referral code (typically their first name in uppercase)
2. Add them to this document
3. Share the links with them

No code changes needed - the system automatically supports any referral code!
